Programming with Inexpensive Robots
Coursework

C, Arduino, Java

This project-based course focused on the gradual implementation of several robotic algorithms given the constraints of a $100 children's learning robot with limited RAM. Features implemented included: probability based mapping, edge detection, obstacle avoidance, localization, and ultimately SLAM (Simultaneous localization and mapping). A Bluetooth module attached to the robot was leveraged to mitigate memory constraints and utilize a remote terminal for more complex tasks, sending commands and receiving location and mapping data from the robot. The map and more complex routines, including the blue tooth receiving module, was programmed in Java and displayed using Java's Swing GUI package, while simple routines and actions were programmed in C on the robot itself.
